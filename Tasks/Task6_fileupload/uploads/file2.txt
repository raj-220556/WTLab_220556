--- Visualization ---

matplotlib.pyplot 
When you use plt, you are usually working with two main components:
 - The Figure: The overall window or page where everything is drawn.
 - The Axes: The area where the data actually appears (the plot with the $x$ and $y$ lines).

simple example Learned To Understand:
---- Simple graph Ploting ----
	import matplotlib.pyplot as plt	
	
	# 1. Define Data
	x = [1, 2, 3, 4]
	y = [10, 20, 25, 30]
	
	# 2. Create the plot type
	plt.plot(x, y, color='green', marker='o')
	
	# 3. Customize (Labels and Title)
	plt.title("Sales Over Time")
	plt.xlabel("Months")
	plt.ylabel("Revenue")
	
	# Display Result
	pltshow()

---- Histogram -----
a Histogram takes a single numerical column and groups the data into "bins" (ranges) to show you where most of your values fall.

- bins : Is used to group the values ranges
import matplotlib.pyplot as plt

# Using the 'age' column from your dataset
plt.hist(df['age'])
plt.title("Distribution of Age")
plt.xlabel("Age")
plt.ylabel("Frequency")
plt.show()

5. Heatmap – Correlation Between Variables

Syntax used

sns.heatmap(data.corr(), annot=True, cmap='coolwarm')



data.corr()

→ Computes correlation matrix.annot=True

→ Shows correlation numbers on the map.

cmap='coolwarm'

→ Color theme of heatmap.



Correlation values:

+1 → strong positive relation

0 → no relation

-1 → strong negative relation
